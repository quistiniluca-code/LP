# ECON Lead Generation Landing

Landing statica ECON by APEX orientata alla lead generation.

## File principali

- `index.html` — landing page completa, mobile-first, con form Netlify.
- `thank-you.html` — pagina di ringraziamento e conversion tracking.
- `_redirects` — redirect Netlify.
- `netlify.toml` — configurazione build statica.

## Deploy Netlify

Build command: nessuno.
Publish directory: `.`
Branch: `main`.

## Form Netlify

Nome form: `analisi-energetica`.

Campi principali:
- nome
- telefono
- email
- azienda
- comune
- tipo_immobile
- spesa_mensile
- bolletta
- messaggio
- privacy

## Da completare

- Inserire ID Google Tag Manager reale.

- Inserire link privacy definitivo.
- Eventualmente sostituire il marchio CSS con il logo ufficiale ECON.
- Collegare il repository a Netlify oppure usare deploy manuale.
